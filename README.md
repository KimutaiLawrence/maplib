# MapLib

A high-performance mapping library built from scratch for modern geospatial applications.

## Design Goals

- **Speed**: Optimized rendering with WebGL for fast, smooth maps
- **AI Integration**: Built-in support for AI-driven geospatial workflows
- **Modern Stack**: TypeScript-first, reactive API design
- **Minimal Dependencies**: Lean core with optional plugins

## Features (Planned)

- Vector tile rendering
- Raster tile overlays
- GeoJSON support
- Interactions (pan, zoom, rotate)
- Layer management
- Geolocation
- Custom projections
- Real-time data updates

## Getting Started

```bash
npm install
npm run dev
```
